# Birds-Eye-View-Optimization-Tool

How to properly configure Birds-Eye-View-Optimization-Tool:
1. Open Visual Studio 2019+
2. Select Clone a Repository
3. Copy https://github.com/gregolex/Birds-Eye-View-Optimization-Tool
4. Paste into Repository Location
5. Change Local Path if desired
6. Select Clone
7. After cloning is complete select view and select Main.vb

Birds Eye View Optimization Tool uses send keys to automated processes automatically in the same steps a technician would use to optimize a computer system.
